# Code of Conduct

This project expects respectful, technical collaboration focused on reproducible formal work.

## Expected behavior

- discuss the mathematics and the code directly
- keep criticism specific and professional
- respect review boundaries and repository scope

## Unacceptable behavior

- harassment, abuse, or discriminatory conduct
- personal attacks
- disruptive behavior that blocks normal collaboration

Project maintainers may edit, remove, or reject contributions that violate these standards.
